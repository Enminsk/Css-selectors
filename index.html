27 CSS селекторов

№.1 Селектор * В css свойствах с помощью этого селектора установлены font-family и у всех элементов удалены дефолтные margin и padding
* { font-family: 'Open Sans', sans-serif; margin: 0; padding: 0; }
№.2 Селектор вида #X Знак решетки перед CSS-селектором Х выделит нам элемент с id = Х.
#id-example { background-color: antiquewhite; }
№.3 Селектор вида .X Это CSS-селектор класса Х. Разница между id и классом заключается в том, что одному классу может принадлежать несколько элементов на странице. Используйте классы, когда вы хотите применить стиль к нескольким однотипным элементам.
.class-example { box-shadow: 5px 5px 5px black; }
№.4 Селектор вида X Y CSS-селектор дочерних элементов встречается чаще всего. Если вам надо выделить элементы определенного типа из множества дочерних элементов, используете этот селектор. Например, вам надо убрать линию из всех ссылок которые находятся в классе .links. В этом случае используйте этот селектор.
.links a { text-decoration: none; }

№.5 Селектор вида X Если вы хотите охватить все элемента одного тега используйте селектор по тегу
section, header { border: 1px solid; margin: 8px; padding: 8px; display: flex; flex-direction: column; }
№.6 Селектор вида X:visited и X:link Мы используем псевдо-класс :link, для выделения всех ссылок, на которые еще не кликнули. Если же нам надо применить определенный стиль к уже посещенным ссылкам, то используем псевдо-класс :visited.

Важно! Ваш браузер запоминает какие ссылки вы посещали, если вы применили селектор :link и он не изменил цвет, значит вы уже посещали эту страницу.

a:link { color: red; } a:visited { color: green; }

№.13 Селектор вида X[attr^=value] Выбирает элементы с атрибутом attr (его имя — это значение в квадратных скобках), значение которого начинается с value.
.link-with-attr-start-value a[href^="https://"] { color: red; }

№.15 Селектор вида X[data-*="foo"] Как же нам теперь написать CSS-селектор, который бы выделял ссылки на все виды изображений? Например, мы могли бы написать так:
a[href$=".JPG"], a[href$=". JPEG"], a[href$=".PNG"], a[href$=".GIF"] { color: red; }
Но это геморрой и неэффективно. Другим возможным решением является применение пользовательских атрибутов. Почему бы нам не добавить наш собственный атрибут data-filetype в каждую ссылку? Теперь мы можем выделить такие ссылки при помощи данного CSS-селектора:
.link-with-data-type a[data-filetype="image"] { color: red; }

№.16 Селектор вида X[foo~="bar"] Тильда (~) позволяет выделить определенный атрибут из списка атрибутов, разделенных запятой.
.link-with-some-data-type a[data-info~="external"] { color: red; } .link-with-some-data-type a[data-info~="image"] { border: 1px solid black; }

№.17 Селектор вида X:checked Этот псевдокласс выделяет только элементы пользовательского интерфейса, такие как переключатель или флажок. Причем те, которые отмечены/выбраны.
.input-checked input[type="radio"]:checked { border: 6px solid black; }

№.21 Селектор вида X:nth-last-child(n) :nth-last-child(an+b) находит элемент, имеющий an+b-1 потомков после данной позиции в дереве документа, значение для n может быть положительным или нулевым, а также имеющий родительский элемент.
